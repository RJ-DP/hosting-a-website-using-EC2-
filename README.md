
# Hosting a Website on AWS EC2 
## Steps to Deploy
### 1. Launch EC2 Instance
* Go to AWS Console, then EC2, then Launch Instance
* Choose Ubuntu
* Create and Use key pair
* Use Putty
* Allow HTTP (80), HTTPS (443), and SSH (22)
* Launch instance
